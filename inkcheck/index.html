Inkcheck 0.7 · A free community tool

Inkcheck

Open-source mechanical QA for stories made with ink: bounded, repeatable checks with exact paths to problems. Inkcheck does not use AI to read, test, or rewrite your story.

1 Test your story

What it does: Inkcheck automatically explores your story .ink file logic, tries different choices, and produces a report of broken paths or places it could not reach to help automate quality assurance.

Coverage you can trust. Inkcheck combines systematic search with seeded sampling, records the exact path to each problem, streams real work progress, and labels partial results. A quiet run is never presented as proof that it checked everything.

Need a power-user offline solution? Inkcheck 0.7 can also run locally from npm with this command:

      

      npx -y inkcheck path/to/story.ink
      
This hosted checker explores up to 1,000,000 story states per run and marks the report partial when it stops there. The local tool defaults to 10,000,000 states, supports a ceiling of 100,000,000, streams progress, and adds durable campaigns for longer human or agent workflows. Agents can drive the same mechanical checks through MCP; Inkcheck itself still does not use AI.

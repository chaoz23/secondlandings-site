Inkcheck

An independent, open-source second pair of eyes for stories made with ink, inkle’s language for interactive narrative.

1 Test your story

What it does: Inkcheck automatically explores your story .ink file logic, tries different choices, and produces a report of broken paths or places it could not reach to help automate quality assurance.

Repeatable, bounded coverage. Inkcheck combines systematic search with seeded sampling, records the path to each problem, and tells you when a run was partial rather than pretending it proved everything.

Support Inkcheck ♥

Need a power-user offline solution? Inkcheck can also run locally from npm with this command:

npx -y inkcheck path/to/story.ink

This checker explores up to 1,000,000 story states per run — plenty for most stories. For a big or deeply branching story that needs a broader pass, the local command-line tool goes up to 100,000,000 states, lets you control memory, and streams progress as it runs.
